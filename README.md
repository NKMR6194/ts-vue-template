# ts-vue-template

A Vue + TypeScript + Webpack setup


## Setup

1. Change dir name
1. Edit package.json
    `name`,
    `version`,
    `description`,
    `author`,
    `licence`
1. Edit LICENSE
1. Edit README.md
1. Edit .gitignore

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Main dependencies

- Vue 2.5
- TypeScript 2.7
- Webpack 4.1
